# Logic Challenge - Palindrome

## Problem

## English

|   Bahasa                              | English               |

| Diberukah subuah function             | Given a function      |
| yang merina                           | That accepts          |
| function terbesut                     | The function          |
| akan mengembalikan nilai true         | will return value true|
| tepat sama                            | exactly same          |
